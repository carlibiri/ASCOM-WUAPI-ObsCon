ASCOM Observing Conditions Weather Underground API driver : V 1.1.0

Setup requires only 2 steps :

  • Go to https://www.wunderground.com/ and select your desired weather reporting station. Copy that station's Station ID from its page or the URL

  • Launch the driver configuration page from your client before connecting, and enter that station ID into the appropriate box. You'll only have to do this once.

Once configured, connect to the driver, and enable whatever view your client offers for observing conditions. The driver attempts to refresh data every 30s. Actual refresh rate is dependent on the submitting station.
